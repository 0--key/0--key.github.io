Testing Python primary constants

Now it is time to concoct our first primitive set of tests to determine the existence of Python primals True, False and None. It is intentionally simplified to show you the threshold of primitivity in valid Python code. 

import unittest


class TestLanguagePrimals(unittest.TestCase):

    def test_does_true_ever_exist(self):
        self.assertTrue(True)

    def test_does_false_ever_exist(self):
        self.assertFalse(False)

    def test_does_none_ever_exist(self):
        self.assertIsNone(None)
